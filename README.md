### The Workshop

*   **[Gostman](https://github.com/krockxz/gostman)** `Go` `React` `Wails`
    **A 10x lighter, zero-telemetry Postman alternative.** A native, local-first API client for REST, GraphQL, and WebSockets. Built for speed, not bloat.

*   **[Codebase-indexer](https://github.com/krockxz/codebase-indexer)** `Rust` `Tantivy` `ONNX` `Tree-sitter`
    **Cursor-style semantic code search, entirely offline.** A blazingly fast CLI that parses ASTs and fuses BM25 with local vector embeddings to find exactly what you need.

*   **[Un-nexted](https://github.com/krockxz/Un-nexted)** `React` `Node.js` `SSR Architecture`
    **Deconstructing the magic of modern frameworks.** A bare-metal server-side rendering pipeline built from scratch to expose how bundling, routing, and hydration actually work.

*   **[TaskFlow](https://github.com/krockxz/TaskFlow)** `Next.js` `Supabase` `PostgreSQL`
    **Self-hosted async coordination.** A real-time task management hub for distributed teams, locked down by strict database-level Row Level Security and tightly synced with GitHub.

*   **[MailFlowAI](https://github.com/krockxz/MailFlowAI)** `AI/LLMs` `Workflow Automation`
    **High-performance inbox intelligence.** An AI-native email assistant designed to ruthlessly eliminate busywork and automate professional communication.

*   **[pdf2docx-healer](https://github.com/krockxz/pdf2docx-healer)** ([PyPI](https://pypi.org/project/pdf2docx-healer/)) `Python` `pdf2docx` `python-docx`
    **A drop-in replacement for pdf2docx that actually preserves your formatting.** A PyPI-published package that heals bullet lists, hyperlinks, CJK fonts, and scanned PDFs in a post-processing pass. Install with `pip install pdf2docx-healer`.
